EngMod4T

 

Engmod4T was developed by

 

Neels van Niekerk

of

Vannik Racing Developments

 

EngMod4T is a 1-dimensional gasdynamic multi cylinder four stroke engine simulation program. This Help-system describes the required input variables and options available. The input files are prepared by using the pre-processor Dat4T. Also described are the output files and the folder structure.

EngMod4T is a licensed commercial software package and it requires an authorization file before it will run.

The simulation models and simulation methods used in EngMod4T are mostly based on the work of Professor Emeritus Gordon P Blair of the Queens University of Belfast.
