Modeling of Boxes and Plenums

 The terminology of a “ box” and a “plenum” is both used to describe a container of a fixed volume that is connected to one or more pipes in the exhaust system. The user has to specify the box volume in cm3 and the pipe end configuration at the entry or exit from the box. The user has a choice between a sharp edge or a rounded edge for the pipe end.
