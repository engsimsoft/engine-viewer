[EngMod4T is a 1-dimensional gasdynamic multi cylinder four stroke engine simulation program. This Help-system describes the required input variables and options available. The input files are prepared by using the pre-processor Dat4T. Also described are the output files and the folder structure.]{lang="EN-US" style="font-size:12.0pt"}

[EngMod4T is a licensed commercial software package and it requires an authorization file before it will run.]{lang="EN-US" style="font-size:12.0pt"}

[The simulation models and simulation methods used in EngMod4T are mostly based on the work of [Professor Emeritus Gordon P Blair](References.htm) of the Queens University of Belfast.]{lang="EN-US"}
