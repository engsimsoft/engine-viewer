## [Installation]{.underline}

[Create a folder, typically called EngMod4T, and copy all the files to this folder. The files should be: ]{lang="EN-US" style="font-size:12.0pt"}

1.  Dat4T.exe
2.  Dat4THelp.chm
3.  EngMod4T.exe
4.  EngMod4THelp.chm
5.  EngMod4T.fig
6.  Post4T.exe
7.  Post4T.chm

If this is the first time you are using the software and it has not yet been authorised to run on your computer you will not yet have the last file, EngMod4T.fig. [Authorization](Authorization.htm) explains how to get this file.
