# [Importing Measured Traces]{.underline} {#importing-measured-traces align="left"}

## Importing Measured Data from File {#importing-measured-data-from-file align="left"}

It is possible to import an ASCII file which has degrees as its first
column and pressure as its second column by selecting \"Import Measured
Trace File\" menu item on the drop down menu after which the following
dialog will open:

![](10-Pictures/MeasuredTraceImport.jpg){border="0" width="818"
height="500"}

The imported file can have any extension but must be an ASCII file.
**Only the first two columns will be imported**

## General Information {#general-information align="left"}

The user has to select the following:

- 
- Whether the input data is in kW or hp.
- Between a naturally aspirated or turbocharged engine.
- Whether the input data is in the inlet system, cylinders or exhaust
  system.
- Whether the decimal character is a point or a comma.

The user has to specify the following:

- 
- The valve opening and closing points.
- The number of cylinders.
- The rpm at which the trace was recorded.
- The line number in the input file where the data starts.
